AI驱动的竞品情报自动化服务 | 7维度日报 | 定时推送 | 按月订阅

## 服务描述

### 服务概述

提供基于 AI Agent 的**竞品情报自动化订阅服务**。客户指定目标公司（股票代码），系统每日自动生成一份 7 维度结构化情报报告，通过大象/飞书/企微/邮件定时推送。

### 交付物

每日一份 Markdown/PDF 格式竞品情报报告，包含以下 7 个板块：

1. **核心财务指标快照** — 市值、PE、PEG、营收增速、毛利率、ROE、负债率等
2. **新闻动态** — 当日 Top 10 相关新闻（标题+来源+摘要）
3. **社交媒体舆情** — Twitter/X 热帖汇总（KOL 观点+互动量排序）
4. **机构持仓变动** — 头部基金增减仓方向（Vanguard/BlackRock/Fidelity 等）
5. **内部人交易信号** — CEO/CFO/董事 买入卖出记录（含金额和时间）
6. **分析师预测** — 营收/EPS 一致预期、目标价区间、覆盖分析师数量
7. **AI 深度分析** — 竞品格局变化、潜在催化剂、风险评估（基于 Perplexity Pro 联网分析）

### 技术方案

| 组件 | 技术选型 |
|------|----------|
| 数据采集 | AISA API 聚合网关（Twitter + FMP Financial + Perplexity） |
| 调度引擎 | OpenClaw Cron（每日定时触发，失败自动重试） |
| 报告生成 | Python 脚本（Markdown/JSON 双格式） |
| 推送通道 | 大象/飞书/企微/Telegram/邮件（客户选择） |
| 降级机制 | 单数据源失败不影响整体报告（graceful degradation） |
