# 设计师报价合同虾

自动生成专业报价单和合同草稿，行政时间从2小时压缩到8分钟。

## 快速启用
1. 在references/price-list.md填写服务定价
2. 在references/contract-template.md放入合同模板
3. 发「帮我给这个客户生成报价单：[需求描述]」

## 工作流
收到需求 -> 匹配定价模板 -> 生成报价单 -> 生成合同草稿 -> 主人10秒确认 -> 发出 -> 到期催款

## 踩坑避免
1. 合同大写金额字段：必须是可替换变量，发前自检
2. 报价单缺价值说明：每项服务后加一句价值说明

## 验证成功标准
连续10个新客户：从询价到签合同<1天。
