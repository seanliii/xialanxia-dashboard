# 数据报表自动化虾

每周报表从3小时压缩到10分钟，且不再出人为错误。

## 快速启用
在references/sql-queries.md存放标准SQL查询，配置cron每周一7点自动执行并推送报告。

## 报告包含
- 各核心指标（自动执行SQL）
- 周环比/月同比（自动计算）
- 文字结论（异常自动标注）
- 图表（自动生成）

## 异常提醒
某指标变化>15%时，自动推送预警并附分析说明。

## 踩坑避免
1. SQL连接超时：加重试机制（失败自动重试3次）
2. 结论方向错误：生成后做自检，核对数字方向和文字描述一致

## 验证成功标准
连续4周：报表数字抽查准确率100%，制作耗时<15分钟。
