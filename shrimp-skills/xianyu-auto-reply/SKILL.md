# 闲鱼自动回复接单虾

帮助闲鱼卖家自动处理买家询问，节省每天2-3小时盯手机时间，全天候接单（尤其是晚间和凌晨），月均多接28笔。

## 快速启用

**第一步：填写商品信息**（修改 `references/config.md` 中的配置）

```yaml
# 基础配置
shop_name: 我的闲鱼店铺
product_name: iPhone13充电线
original_price: 29
min_price: 24  # 最低可接受价格
shipping: 顺丰，24小时内发货
```

**第二步：配置Q&A**

编辑 `references/qa-template.md`，添加你的商品常见问题和标准回答。

**第三步：设置cron**

```bash
# 每15分钟扫描一次消息
openclaw cron add --schedule "*/15 * * * *" --message "扫描闲鱼消息并自动回复，遇到成交信号立即推送通知"
```

## 核心工作流

```
买家发消息
    ↓
匹配Q&A库 → 找到匹配 → 自动回复
    ↓ 未匹配
检测成交信号（"好的下单"/"直接拍"/"确认") → 推送通知给主人
    ↓ 非成交信号
记录到待处理列表，每日推送汇总
```

## 踩坑避免

1. **成交信号词设太宽**：只用"好的下单"/"直接拍"/"确认下单"触发通知，"好的"/"嗯"不触发
2. **议价全部拒绝**：在Q&A里加"多次议价时让步1元并送5元优惠券"的弹性规则
3. **同名买家混淆**：用买家ID而非昵称标识用户

## 每日汇报格式

```
📦 闲鱼日报 [日期]
收到询问：X条
已自动回复：X条
待人工处理：X条（见下）
成交信号：X条 ⬅ 重点查看
```

## 验证成功标准

开启1周后对比：晚间10点后成交订单占比从<5%提升到>20%。
