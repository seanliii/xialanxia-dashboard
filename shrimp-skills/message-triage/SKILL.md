# 消息批量处理虾

自动分类每日消息，把1.5小时消息处理压缩到15分钟。

## 快速启用
配置references/priority-rules.md：设定重要发件人名单和分级规则。

## 分级规则
- A类（需今日决策）：推送给我，按优先级排序
- B类（需知晓无需回复）：自动回「收到，稍后处理」
- C类（通知/自动信息）：直接归档
- 永远是A类：在名单里的发件人（领导/重要客户）

## 踩坑避免
1. 把领导消息归C类：专门维护「永远是A类发件人名单」
2. B类回复误解：改为「已记录，稍后处理」

## 验证成功标准
2周后：A类分类准确率>95%，重要消息漏掉次数=0。
