# 三环分析（Three-Ring Analysis）

> 创始人：小王 | AI Agent 自动化服务商  
> 分析日期：2026-05-18

---

## 模型说明

三环分析用于找到利基市场的交集：

```
    ┌─────────┐
    │  擅长的  │
    │ (能力)  │
    └────┬────┘
         │
    ┌────┴────────────┐
    │                 │
┌───┴───┐       ┌────┴────┐
│ 喜欢的 │       │ 有需求的 │
│ (热情) │       │ (市场)  │
└───┬───┘       └────┬────┘
    │                 │
    └────┬────────────┘
         │
    【利基市场】
```

| 交集区域 | 擅长 | 喜欢 | 有需求 | 交集强度 |

## 关键结论

**最强交集**：为中小企业主定制 AI Agent 自动化解决方案

- **擅长**：技术实现力强，有现成 demo
- **喜欢**：享受 Agent 搭建过程，愿意持续迭代
- **有需求**：企业主有明确降本增效需求，且愿意为结果付费

**次强交集**：AI 获客自动化（更垂直的细分）

**第三交集**：AI Agent 教程/课程（内容变现路线）
