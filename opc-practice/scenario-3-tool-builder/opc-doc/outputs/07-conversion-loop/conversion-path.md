## 三阶段详解

### 阶段一：触达（Awareness）

**目标**：让多平台跨境卖家知道"有个工具能帮你一键搞定listing本地化"

| 触达方式 | 核心内容 | 转化到 |
|---------|---------|--------|
| Reddit 痛点讨论帖 | "你们跨平台 listing 翻译花多少时间？" | Landing Page |
| Twitter building in public | 进展分享 + tips | Landing Page |
| PH Coming Soon | 产品概念 + 演示 | Waitlist |
| Lead Magnet | 免费「Listing 本地化 Checklist」PDF | Email 列表 |

**量化目标**：4周内 1000+ 曝光 → 200+ 点击 → 100+ waitlist

---

### 阶段二：承接（Capture & Nurture）

**目标**：把看到的人变成留下联系方式的人，然后通过邮件建立信任

| 承接触点 | 内容 | 目的 |
|---------|------|------|
| Landing Page | 价值主张 + 3秒 demo GIF + "Join Waitlist" | 收集邮箱 |
| Welcome Email | 感谢 + 预计上线时间 + 1个免费 tip | 建立第一印象 |
| Drip Email #1（Day 3） | Case Study：卖家用工具省了多少时间 | 强化价值 |
| Drip Email #2（Day 7） | 痛点共鸣 + 数据支撑 | 加深需求认知 |
| Drip Email #3（Day 10） | Early Access 邀请 + 半价锁定 | 推向成交 |

**量化目标**：waitlist → 打开率 50%+ → 点击率 20%+ → 试用申请 30%+

---

### 阶段三：成交（Conversion）

**目标**：让试用用户变成付费用户

| 成交策略 | 具体做法 | 降低摩擦 |
|---------|---------|---------|
| 半价首月 | $25/月（正价$49） | 经济门槛低 |
| 退款保证 | 14天不满意全退 | 消除风险 |
| 个性化 onboarding | 老张亲自帮导入首批 listing | "代驾式"体验 |
| Quick Win | 首次使用5分钟内看到翻译结果 | 即时价值感 |
| Social Proof | "已有 X 位卖家在用" | 从众效应 |

**量化目标**：试用 → 付费转化 50%+ | 首月 → 第二月续费 70%+
