# 定位声明 — ListingBridge

## 一句话定位

**ListingBridge 是面向多平台跨境电商卖家的 AI Listing 本地化工具，帮助卖家一键将产品 listing 翻译、适配并优化到任意平台和语种，提升跨站转化率。**

---

## 定位公式

```
为 [多平台跨境电商卖家]
解决 [手动翻译 listing 耗时、格式适配复杂、本地化质量差导致转化低] 的问题
通过 [AI 智能翻译 + 平台格式自动适配 + 本地 SEO 关键词优化]
不同于 [大型全套 ERP（贵且重）或通用翻译工具（不懂电商语境）]
我们的差异化是 [电商场景专精 + 轻量独立 + AI-native + 支持多平台格式输出]
```

## 价值主张

### 对用户

| 之前 | 之后 |
|------|------|
| 一个 listing 翻译+适配 3 个平台要 2 小时 | 10 分钟搞定 |
| 用 Google Translate，语法对但不懂电商卖点 | AI 懂产品卖点，翻译即营销 |
| 不同平台标题字数限制/bullet 格式不同，手动调 | 自动适配各平台规则 |
| SEO 关键词靠猜 | 自动本地化高频搜索词 |

### 量化承诺

- 单个 listing 跨 3 平台适配时间：从 2 小时 → 10 分钟（节省 90%）
- 本地化 listing 的点击率提升：预期 15-30%（vs. 通用翻译）

## 产品边界

### 做什么（V1 范围）

- Amazon → Shopify / eBay / Walmart listing 一键转换
- AI 翻译（英/德/法/西/日 5 语种起步）
- 各平台格式自动适配（标题长度、bullet 数量、描述格式）
- 本地 SEO 关键词建议
- 批量处理（CSV 导入/导出）

### 不做什么

- 不做库存/订单管理（那是 ERP）
- 不做广告投放优化（保持聚焦）
- 不做平台 API 直接同步上架（V1 先导出文件，降低技术风险）
- 不做中文市场
