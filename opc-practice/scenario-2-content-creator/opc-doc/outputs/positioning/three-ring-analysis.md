# 三环分析（Three-Ring Analysis）

> 创始人：小李 | 分析日期：2026-05-18  
> 方法论：OPC 一人企业 · 利基定位阶段

---

## 分析框架

三环交叉 = **你擅长的** ∩ **你热爱的** ∩ **市场愿意付费的**

只有三环同时重叠的区域，才是可持续的一人企业利基。

---

## 环一：擅长（能力 × 经验的交叉）

```
核心擅长领域：
├── 结构化思维与方法论输出（产品经理底层能力）
├── 用户研究与需求洞察（特别是女性消费者）
├── 内容创作（文案 + 视频，双栖能力）
├── 数据驱动决策（增长分析、效果归因）
├── 课程/知识体系设计（教育产品经验）
├── 跑步科学训练（教练级）
└── 社群搭建与维护
```

**擅长度排序：**
1. 文案/内容创作 → 直接可交付
2. 用户研究/需求洞察 → 能精准定位痛点
3. 产品方法论体系化 → 能做课程
4. 跑步训练方案设计 → 能做服务
5. 视频制作 → 中级但够用

---

## 环二：热爱（做起来有能量、可持续的事）

```
高能量活动：
├── 帮助职场女性找到节奏感（使命驱动）
├── 跑步本身 + 教人跑步（身体愉悦 + 成就感）
├── 写深度分析文章（心流状态）
├── 做内容并看到反馈（小红书互动）
├── 1对1辅导/咨询（建立连接）
└── 研究用户行为（天然好奇心）

低能量/排斥：
├── ❌ 迎合算法做低质内容
├── ❌ 卖焦虑式营销
├── ❌ 纯技术类写作
└── ❌ 服务男性受众（无共鸣）
```

**热爱度排序：**
1. 帮助同龄职场女性变好 → 人生使命级
2. 跑步相关一切 → 天然热爱
3. 内容创作 → 有正反馈循环
4. 产品方法论 → 有热情但非顶级
5. 数据分析 → 擅长但不热爱

---

## 环三：市场付费意愿（验证过的需求）

| 方向 | 付费人群 | 付费金额参考 | 竞争烈度 | 市场规模 |
|------|---------|-------------|---------|---------|
| 职场女性跑步/运动 | 25-35岁白领女性 | 99-399元/课程 | 中低 | 中 |
| 产品经理求职/晋升 | 1-5年PM | 199-999元 | 高 | 中大 |
| 女性自律/时间管理 | 25-35岁职场女性 | 99-299元 | 中高 | 大 |
| 小红书运营方法论 | 新手博主 | 99-499元 | 高 | 中 |
| 跑步训练计划定制 | 跑步爱好者 | 199-599元/期 | 低 | 小 |

**市场机会排序：**
1. 职场女性自律/生活方式 → 大市场、中竞争、高共鸣
2. 女性跑步入门 → 精准细分、低竞争、强差异化
3. 产品经理进阶 → 大市场但红海

### ⭐ 核心交叉区域（三环重叠）

1. **职场女性「用跑步重建生活节奏」的方法论内容 + 服务**
   - 擅长：跑步教练 + 内容能力 + 用户研究
   - 热爱：帮助女性 + 跑步 + 创作
   - 市场：运动消费升级 + 自律类内容付费意愿强

2. **产品经理思维 × 女性生活方式的跨界内容**
   - 擅长：产品方法论 + 文案 + 视频
   - 热爱：写深度内容 + 帮助同龄人
   - 市场：差异化定位，竞争者少

3. **女性跑者社群 + 训练服务**
   - 擅长：社群运营 + 训练方案设计
   - 热爱：跑步 + 1对1连接
   - 市场：私域高客单价服务
