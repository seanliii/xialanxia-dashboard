# B2B线索挖掘 — 复刻方法

## 这是什么
一个帮 B2B 销售/创业者快速找到目标客户的工具。输入行业和地区，自动生成目标公司列表、联系方式获取策略、冷邮件模板。

## 适合谁
- 做 B2B 业务找客户太费时间的创业者
- 想系统化获客但不想花钱买数据的销售
- 做 AI 工具想找企业客户的独立开发者

## 复刻步骤

### Step 1：创建仓库
新建 GitHub 仓库 b2b-lead-hunter，开启 Pages

### Step 2：核心功能
- 行业选择器：预设热门行业分类
- 地区筛选：城市/省份/国家
- 线索生成逻辑：基于输入生成搜索策略
- 邮件模板库：冷触达/跟进/合作邀请
- 联系方式获取方法指引

### Step 3：关键 Prompt

```
做一个 B2B 线索挖掘工具页面：
- 输入区：目标行业（下拉选择）、目标地区、公司规模、关键词
- 输出区：搜索策略建议、LinkedIn搜索语法、邮件模板（可一键复制）
- 附加：冷邮件最佳实践、跟进节奏建议
- 科技感深色主题，卡片布局
```

### Step 4：部署
推送代码，GitHub Pages 自动上线

## 踩坑记录
- 搜索策略要具体可执行，不能太泛
- 邮件模板避免垃圾邮件触发词
- 联系方式获取要合规，不能教爬取
