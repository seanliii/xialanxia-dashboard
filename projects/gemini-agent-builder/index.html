📖 什么是 Managed Agents？

Gemini API Managed Agents 让你用一次 API 调用启动一个运行在隔离 Linux 沙箱的 Agent，它可以推理、执行代码、管理文件、浏览网页——完全自主。

最酷的部分：Agent 行为用 AGENTS.md 和 SKILL.md 文件定义，可版本化管理——这和 OpenClaw 完全相同的架构哲学！

🔑 快速开始

curl https://generativelanguage.googleapis.com/v1beta/agents:run \
  -H "x-goog-api-key: YOUR_GEMINI_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3.5-flash",
    "message": "Research AI agent trends",
    "agentConfig": {
      "agentsmd": "...",
      "skills": [...]
    }
  }'
