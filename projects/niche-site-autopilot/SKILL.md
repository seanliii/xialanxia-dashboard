# 利基网站自动运营 — 复刻方法

## 这是什么
一个帮你从零开始做 SEO 利基站的自动化工具。从选利基到关键词研究到内容生成到 SEO 优化，全流程自动化引导。

## 适合谁
- 想做 SEO 站赚广告费但不知道选什么利基的人
- 会写代码但不懂 SEO 的开发者
- 想建立长期被动收入但不知道第一步怎么走

## 复刻步骤

### Step 1：创建仓库
新建 GitHub 仓库 niche-site-autopilot，开启 Pages

### Step 2：核心功能模块
- 利基选择器：输入兴趣领域 → 推荐具体利基方向
- 关键词生成器：基于利基自动生成长尾关键词列表
- 内容大纲生成：每个关键词自动生成文章大纲
- SEO 评分卡：检查标题、描述、关键词密度
- 竞争分析建议：如何评估关键词难度

### Step 3：关键 Prompt

```
做一个利基网站自动运营系统页面：
- 第一步：输入兴趣标签，推荐3-5个具体利基方向（含搜索量预估）
- 第二步：选定利基后，自动生成20个长尾关键词
- 第三步：每个关键词生成文章大纲（标题+H2+要点）
- 第四步：SEO优化检查清单（可勾选）
- 现代暗色UI，进度条连接各步骤
```

### Step 4：部署
推送到 main 分支，GitHub Pages 自动部署

## 踩坑记录
- 利基选择要够小，「健身」太大，「居家哑铃训练 for 程序员」才对
- 长尾关键词要验证搜索量，不能瞎编
- SEO 规则经常变，内容要定期更新
