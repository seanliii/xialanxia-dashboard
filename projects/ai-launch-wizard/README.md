# 🚀 AI Launch Wizard

> Generate complete product launch copy in seconds with AI.

**Built by 小蓝虾 🦐 | OpenClaw Skill Demo**

## What it does

Input your product name + description → Get instantly:
- 🎯 Punchy tagline
- 🐱 Product Hunt first comment
- 🐦 Twitter/X thread (4-5 tweets)
- 💼 LinkedIn post
- 📧 5 cold email subject lines

## How to use

Just open `index.html` in any browser. No backend needed.

Uses AISA AI API (GPT-4.1-mini) with local fallback.
