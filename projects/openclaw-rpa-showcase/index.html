1
🎥 用真实Chrome录制操作
在OpenClaw中告诉它"帮我录制在XXX网站提交报销表单的操作"，AI用浏览器录制你的操作流程，选择器直接从DOM读取（不靠模型猜），录制时会消耗少量token，但只需录一次。
User: 帮我录制：登录 erp.company.com → 提交报销 → 下载审批记录 Agent: 正在录制...识别到 12 个操作步骤，选择器确认 ✓
2
⚙️ AI编译成独立.py脚本
录制完成后，Agent自动生成可独立运行的 Playwright Python 脚本。支持多步任务拆解，防止单次请求超时。生成后脱离OpenClaw也能跑。
# 生成的 rpa_expense_report.py from playwright.sync_api import sync_playwright def run(): with sync_playwright() as p: browser = p.chromium.launch() page = browser.new_page() page.goto("https://erp.company.com/login") page.fill("#username", "{{USERNAME}}") # ... 12步全自动，零token ✓
3
🔄 零token永久回放
脚本生成后，每次执行完全不需要调用LLM。可以配合OpenClaw cron定时触发，也可以单独运行。速度比AI驱动快100倍，成本降为零。
# 每天9点自动提交报销 → 0 token cost # openclaw cron: "0 9 * * *" → python3 rpa_expense_report.py # 或直接命令行：python3 rpa_expense_report.py --date=today
