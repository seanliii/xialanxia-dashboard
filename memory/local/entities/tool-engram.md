# tool-engram

**Type:** tool
**Updated:** 2026-03-20T07:53:17.372Z

## Facts

- Manages its own memory diary system separately from local memory files.
- Requires periodic triggering of memory_summarize task to update its memory index.
- Provides daily scene/session conversation summaries for memory visualization on the Dashboard.
- 'engram-day-summary' cron job runs daily at 23:47 to generate memory summaries.
- Memory dashboard views rely on session summary files, not on raw memory markdown file storage.
