Provider Tab过滤失效。点击"OpenAI"后，左侧模型按钮区正确过滤为8个OpenAI模型，但右侧对比表格仍显示全部28个模型。

1. `renderCompare()` 函数直接遍历 `MODELS` 全量数组，未使用 `getFilteredModels()`
2. `filterProvider()` 函数只调用了 `renderModelGrid()`，未触发 `renderCompare()` 重绘表格

**修复内容**：
```javascript
// Fix 1: renderCompare() 使用 getFilteredModels()
const models = getFilteredModels();
const costs = models.map(m => { ... });

// Fix 2: filterProvider() 触发 renderCompare()
function filterProvider(p, btn) {
  ...
  renderModelGrid();
  renderCompare();  // ← 新增
}
```
